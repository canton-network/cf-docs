# DA.Bifunctor

## Typeclasses

<a id="class-da-bifunctor-bifunctor-68312"></a>

**class** [Bifunctor]()

<a id="class-da-bifunctor-bifunctor-68312"></a>
 p **where**

> A bifunctor is a type constructor that takes
> two type arguments and is a functor in *both* arguments. That
> is, unlike with `Functor`, a type constructor such as `Either`
> does not need to be partially applied for a `Bifunctor`
> instance, and the methods in this class permit mapping
> functions over the Left value or the `Right` value,
> or both at the same time.

> It is a bifunctor where both the first and second
> arguments are covariant.

> You can define a `Bifunctor` by either defining bimap or by
> defining both first and second.

> If you supply bimap, you should ensure that:

> ```daml
> `bimap identity identity` ≡ `identity`
> ```

> If you supply first and second, ensure:

> ```daml
> first identity ≡ identity
> second identity ≡ identity
> ```

> If you supply both, you should also ensure:

> ```daml
> bimap f g ≡ first f . second g
> ```

> By parametricity, these will ensure that:

> ```daml
> bimap  (f . g) (h . i) ≡ bimap f h . bimap g i
> first  (f . g) ≡ first  f . first  g
> second (f . g) ≡ second f . second g
> ```

> <a id="function-da-bifunctor-bimap-49864"></a>

> [bimap]()
> : : (a -> b) -> (c -> d) -> p a c -> p b d
>   <br/>
>   Map over both arguments at the same time.
>   <br/>
>   ```daml
>   bimap f g ≡ first f . second g
>   ```
>   <br/>
>   Examples:
>   <br/>
>   ```daml
>   >>> bimap not (+1) (True, 3)
>   (False,4)
>   <br/>
>   >>> bimap not (+1) (Left True)
>   Left False
>   <br/>
>   >>> bimap not (+1) (Right 3)
>   Right 4
>   ```

> <a id="function-da-bifunctor-first-18253"></a>

> [first]()
> : : (a -> b) -> p a c -> p b c
>   <br/>
>   Map covariantly over the first argument.
>   <br/>
>   ```daml
>   first f ≡ bimap f identity
>   ```
>   <br/>
>   Examples:
>   <br/>
>   ```daml
>   >>> first not (True, 3)
>   (False,3)
>   <br/>
>   >>> first not (Left True : Either Bool Int)
>   Left False
>   ```

> <a id="function-da-bifunctor-second-99394"></a>

> [second]()
> : : (b -> c) -> p a b -> p a c
>   <br/>
>   Map covariantly over the second argument.
>   <br/>
>   ```daml
>   second ≡ bimap identity
>   ```
>   <br/>
>   Examples:
>   <br/>
>   ```daml
>   >>> second (+1) (True, 3)
>   (True,4)
>   <br/>
>   >>> second (+1) (Right 3 : Either Bool Int)
>   Right 4
>   ```
