---
title: "ThrottleFilterEvaluator"
description: "Throttles logging events, so that no more than maxLoggedOverPeriod unique events are logged over expiryPeriod. De-duplicates logging events over expiryPeriod. Used to ensure that the interactive console remains usable even when the system is in a bad state and logging many errors. Inspired by https://github.com/RotterdamLogisticsLab/logback-throttle/blob/master/src/main/kotlin/com/portofrotterdam/dbs/public/logback/throttle/Throttle.kt"
---
