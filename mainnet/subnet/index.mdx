# Subnet

This site describes how to deploy and run your own subnet of the Canton Network.

The table of contents on the left contains links to tutorials for those getting
started with running a Synchronizer, as well as how-tos, explanations, and
reference materials to assist experienced operators.

You can find more information on Synchronizers
at the [Global Synchronizer Foundation](https://sync.global/docs/), an
independent governance body that oversees the operation of the Global
Synchronizer.
