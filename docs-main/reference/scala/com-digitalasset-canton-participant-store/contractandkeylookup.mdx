---
title: "ContractAndKeyLookup"
description: "Find a contract with the given key. Typically used for Daml interpretation in Phase 3, where the key resolution is provided by the submitter. Returns scala.None$ if the key is not supposed to be resolved, e.g., during reinterpretation by Daml Engine. Returns scala.Some$(scala.None$) if no contract with the given key can be found."
