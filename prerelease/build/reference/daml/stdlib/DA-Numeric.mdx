# DA.Numeric

## Data Types

<a id="type-da-numeric-roundingmode-53864"></a>

**data** [RoundingMode]()

<a id="type-da-numeric-roundingmode-53864"></a>

> Rounding modes.

> <a id="constr-da-numeric-roundingup-99831"></a>

> [RoundingUp]()

> <a id="constr-da-numeric-roundingup-99831"></a>

> > Round away from zero.

> <a id="constr-da-numeric-roundingdown-63242"></a>

> [RoundingDown]()

> <a id="constr-da-numeric-roundingdown-63242"></a>

> > Round towards zero.

> <a id="constr-da-numeric-roundingceiling-67218"></a>

> [RoundingCeiling]()

> <a id="constr-da-numeric-roundingceiling-67218"></a>

> > Round towards positive infinity.

> <a id="constr-da-numeric-roundingfloor-71675"></a>

> [RoundingFloor]()

> <a id="constr-da-numeric-roundingfloor-71675"></a>

> > Round towards negative infinity.

> <a id="constr-da-numeric-roundinghalfup-22532"></a>

> [RoundingHalfUp]()

> <a id="constr-da-numeric-roundinghalfup-22532"></a>

> > Round towards the nearest neighbor unless both neighbors
> > are equidistant, in which case round away from zero.

> <a id="constr-da-numeric-roundinghalfdown-91305"></a>

> [RoundingHalfDown]()

> <a id="constr-da-numeric-roundinghalfdown-91305"></a>

> > Round towards the nearest neighbor unless both neighbors
> > are equidistant, in which case round towards zero.

> <a id="constr-da-numeric-roundinghalfeven-70729"></a>

> [RoundingHalfEven]()

> <a id="constr-da-numeric-roundinghalfeven-70729"></a>

> > Round towards the nearest neighbor unless both neighbors
> > are equidistant, in which case round towards the even neighbor.

> <a id="constr-da-numeric-roundingunnecessary-42017"></a>

> [RoundingUnnecessary]()

> <a id="constr-da-numeric-roundingunnecessary-42017"></a>

> > Do not round. Raises an error if the result cannot
> > be represented without rounding at the targeted scale.

## Functions

<a id="function-da-numeric-mul-81896"></a>

[mul]()
: : [NumericScale](Prelude.md#class-ghc-classes-numericscale-83720) n3 => [Numeric](Prelude.md#type-ghc-types-numeric-891) n1 -> [Numeric](Prelude.md#type-ghc-types-numeric-891) n2 -> [Numeric](Prelude.md#type-ghc-types-numeric-891) n3
  <br/>
  Multiply two numerics. Both inputs and the output may have
  different scales, unlike `(*)` which forces all numeric scales
  to be the same. Raises an error on overflow, rounds to chosen
  scale otherwise.

<a id="function-da-numeric-roundnumeric-41344"></a>

[roundNumeric]()
: : [NumericScale](Prelude.md#class-ghc-classes-numericscale-83720) n => [Int](Prelude.md#type-ghc-types-int-37261) -> [RoundingMode]()
  <br/>
  <a id="type-da-numeric-roundingmode-53864"></a>
   -> [Numeric](Prelude.md#type-ghc-types-numeric-891) n -> [Numeric](Prelude.md#type-ghc-types-numeric-891) n
  <br/>
  Round a `Numeric` number. The value of `round n r a` is the value
  of `a` rounded to `n` decimal places (i.e. scale), according to the rounding
  mode `r`.
  <br/>
  This will fail when using the `RoundingUnnecessary` mode for a number that cannot
  be represented exactly with at most `n` decimal places.
