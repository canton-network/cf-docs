# Connect and build on the Canton Network

Connect to and build regulatory-grade composable applications using Digital Asset’s platform for the Canton Network. The platform includes the Daml smart contract language and SDK, enterprise-grade Canton nodes and synchronizers, and composable modules providing everything you need to deliver applications across a variety of financial use cases.

Learn about the Canton Network and discover how to use applications and assets on the network, deliver your own tokenization and asset mobility use cases on the network, deploy and operate nodes, or independently extend, build, and deploy custom applications and sub-networks.
