title: "HandlerResult"
description: "The given com.digitalasset.canton.lifecycle.FutureUnlessShutdown is an asynchronous processing part for the event. It can run in parallel with any of the following: * Earlier events' asynchronous processing * Later events' synchronous and asynchronous processing * Later stages of synchronous processing for the same event The event will be marked as clean only after the future has completed successfully with com.digitalasset.canton.lifecycle.UnlessShutdown.Outcome. If the future fails with an exception or returns com.digitalasset.canton.lifecycle.UnlessShutdown.AbortedDueToShutdown, the sequencer client will eventually close the subscription."
