---
title: "BlockTransferResponse"
description: "A thin wrapper for a commit certificate. As long as it merely contains a commit certificate (that in turn includes signed and verified data), its signature verification can be safely skipped. As a result, any node can help with state transfer (even when sending responses signed with a new/rotated key)."
