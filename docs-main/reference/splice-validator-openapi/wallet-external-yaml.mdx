**`POST`**

- Operation ID: `createTransferOffer`
- Description: Create an offer to directly transfer a given amount of Amulet to another party. Direct transfers are a three-step process: 1. The sender creates a transfer offer 2. The receiver accepts the offer 3. The sender's wallet automation consumes the accepted offer and transfers the amount.    Amulets are not locked for direct transfers.    If the sender's wallet does not have enough Amulet to fulfill the offer at this point,    the transfer will fail.
- Tags: `wallet`

**Request Body**

- Required: `yes`

| Content Type | Schema | Required Fields |
| --- | --- | --- |
| `application/json` | `object` | `amount`, `description`, `expires_at`, `receiver_party_id`, `tracking_id` |

```json
{
  "receiver_party_id": "<string>",
  "amount": "<string>",
  "description": "<string>",
  "expires_at": "<integer>",
  "tracking_id": "<string>"
}
```

| Code | Description | Content Types | Schemas |
| --- | --- | --- | --- |
| `200` | The transfer offer has been created. | `application/json` | `application/json:object` |
| `400` | Invalid request, check the error response for details. | `application/json` | `application/json:object` |
| `404` | The submitter’s wallet could not be found. | `application/json` | `application/json:object` |
