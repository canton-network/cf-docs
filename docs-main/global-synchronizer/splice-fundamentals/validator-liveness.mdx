In each round, each live validator records its liveness activity with a [ValidatorLivenessActivityRecord](/sdks-tools/api-reference/splice-daml/splice-amulet/splice-validatorlicense#type-splice-validatorlicense-validatorlivenessactivityrecord-17293).

[CIP-0096](https://github.com/canton-foundation/cips/blob/main/cip-0096/cip-0096.md) discontinued validator liveness rewards, phasing the per-validator liveness reward cap down to $0 effective 2026-04-30. Validators now earn CC only through activity rewards -- see [Canton Coin and the Global Synchronizer](/overview/understand/canton-coin) for how validators earn CC today.
