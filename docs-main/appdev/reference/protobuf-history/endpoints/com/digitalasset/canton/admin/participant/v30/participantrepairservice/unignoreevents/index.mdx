```protobuf
rpc ParticipantRepairService.UnignoreEvents(com.digitalasset.canton.admin.participant.v30.UnignoreEventsRequest) returns (com.digitalasset.canton.admin.participant.v30.UnignoreEventsResponse);
```

Remove the ignored status from sequenced events
 This command has no effect on ordinary (i.e., not ignored) events and on events that do not exist.
