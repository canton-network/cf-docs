# Rotate keys with KMS

Canton supports both [rotating the wrapper key](rotate_wrapper_key.mdx#rotate-wrapper-key) when using envelope encryption,
and [rotating Canton keys stored externally in a KMS](rotate_external_keys.mdx#rotate-external-keys).

For smooth disaster recovery, key rotation must be interleaved with backups as described under [Backup and Restore](../../../recover/backup-restore.mdx#backup-restore-private-key-rotation).

* [Rotate the envelope wrapper key](rotate_wrapper_key.mdx)
* [Rotate external KMS keys](rotate_external_keys.mdx)
