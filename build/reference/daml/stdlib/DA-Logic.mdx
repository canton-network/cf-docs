## Functions

<a id="function-da-logic-ampampamp-55265"></a>

: : [Formula]()
  <br/>

  `&&&` is the ∧ operation of the boolean algebra of formulas, to

  `|||` is the ∨ operation of the boolean algebra of formulas, to
  be read as "or"

[true]()
: : [Formula]()
  <br/>
  <a id="type-da-logic-types-formula-34794"></a>
   t
  <br/>
  `true` is the 1 element of the boolean algebra of formulas,
  represented as an empty conjunction.

<a id="function-da-logic-false-99028"></a>

[false]()
: : [Formula]()
  <br/>
  <a id="type-da-logic-types-formula-34794"></a>
   t
  <br/>
  `false` is the 0 element of the boolean algebra of formulas,
  represented as an empty disjunction.

  <a id="type-da-logic-types-formula-34794"></a>

  formulas.

<a id="function-da-logic-substitute-65872"></a>

[substitute]()
: : (t -> [Optional](Prelude.mdx#type-da-internal-prelude-optional-37153) [Bool](Prelude.mdx#type-ghc-types-bool-66265)) -> [Formula]()
  <br/>
  <a id="type-da-logic-types-formula-34794"></a>
   t -> [Formula]() t
  <br/>
  `substitute` takes a truth assignment and substitutes `True` or
  `False` into the respective places in a formula.

<a id="function-da-logic-reduce-40218"></a>

[reduce]()
: : [Formula]()
  <br/>
  <a id="type-da-logic-types-formula-34794"></a>
   t -> [Formula]() t
  <br/>
  `reduce` reduces a formula as far as possible by:
  <br/>
  1. Removing any occurrences of `true` and `false`;
  2. Removing directly nested Conjunctions and Disjunctions;
  3. Going to negation normal form.

<a id="function-da-logic-isbool-80820"></a>

[isBool]()
: : [Formula]()
  <br/>
  <a id="type-da-logic-types-formula-34794"></a>
   t -> [Optional](Prelude.mdx#type-da-internal-prelude-optional-37153) [Bool](Prelude.mdx#type-ghc-types-bool-66265)
  <br/>
  `isBool` attempts to convert a formula to a bool. It satisfies
  `isBool true == Some True` and `isBool false == Some False`.
  Otherwise, it returns `None`.

<a id="function-da-logic-interpret-88386"></a>

[interpret]()
: : (t -> [Optional](Prelude.mdx#type-da-internal-prelude-optional-37153) [Bool](Prelude.mdx#type-ghc-types-bool-66265)) -> [Formula]()
  <br/>
  <a id="type-da-logic-types-formula-34794"></a>
   t -> [Either](Prelude.mdx#type-da-types-either-56020) ([Formula]() t) [Bool](Prelude.mdx#type-ghc-types-bool-66265)
  <br/>
  `interpret` is a version of `toBool` that first substitutes using
  a truth function and then reduces as far as possible.
