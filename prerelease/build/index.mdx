# Build

This site describes how to build Canton Network applications.
Read the [Introduction](overview/introduction.md#build-overview-introduction) to learn what these are and how to build them.

The table of contents on the left contains links to tutorials and guides for
those getting started with building Canton Network applications, as well as
how-to guides and reference materials to assist experienced developers.

For smart contract development, please refer to the [Smart contract development](component-howtos/smart-contracts/index.md#build-component-howtos-smart-contracts-index) section,
and make use of the [Digital Asset Package Manager (Dpm)](dpm/dpm.md#dpm) tool to manage your Daml solution effectively.
