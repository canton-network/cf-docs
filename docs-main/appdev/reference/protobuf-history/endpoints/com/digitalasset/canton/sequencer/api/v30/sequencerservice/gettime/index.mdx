## Signature

```protobuf
rpc SequencerService.GetTime(com.digitalasset.canton.sequencer.api.v30.GetTimeRequest) returns (com.digitalasset.canton.sequencer.api.v30.GetTimeResponse);
```

Return a "current" sequencing time such that, when a `SendAsync` operation is
 subsequently called, if sequenced, the sequencing time of the resulting event is guaranteed to
 be later than the sequencing time previously returned by the `GetTime` call.
