title: "ParticipantPruningServiceGrpc.ParticipantPruningServiceBlockingStub"
description: "A stub to allow clients to do synchronous rpc calls to service ParticipantPruningService. Prunes/truncates the \"oldest\" transactions from the participant (the participant Ledger Api Server plus any other participant-local state) by removing a portion of the ledger in such a way that the set of future, allowed commands are not affected. This enables: 1. keeping the \"inactive\" portion of the ledger to a manageable size and 2. removing inactive state to honor the right to be forgotten."
