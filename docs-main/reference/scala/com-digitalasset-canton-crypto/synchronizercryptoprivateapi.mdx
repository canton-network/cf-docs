---
title: "SynchronizerCryptoPrivateApi"
description: "Wraps the CryptoPrivateApi to include static synchronizer parameters, ensuring that during asymmetric decryption, the static synchronizer parameters are explicitly checked. This is crucial because a malicious counter participant could potentially use a downgraded scheme. For other methods, such as key generation, or signing by this (honest) participant, we rely on the synchronizer handshake to ensure that only supported schemes within the synchronizer are used."
---
