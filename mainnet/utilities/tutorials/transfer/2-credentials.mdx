# Credential Preparation for Token Transfer

## Registrar offers credential of token holder to Investor2

Registrar (of BOND) offers a free credential to Investor2 (as a holder of BOND).

| Actor     | Utility Module   |
|-----------|------------------|
| Registrar | CREDENTIAL       |

Select CREDENTIALS on the left navigation. Click OFFER FREE CREDENTIAL. A window pops up for input.

- Holder: Investor2’s Party ID
- Id: Investor2-BOND-holder
- Description: <anything>
- Claims
  \* Subject: Investor2’s Party ID
  \* Property: isHolderOf
  \* Value: BOND

![image](tutorials/transfer/pictures/token-credentials-4.png)

## Investor2 accepts credential offer

Investor2 accepts the credential offer.

| Actor     | Utility Module   |
|-----------|------------------|
| Investor2 | CREDENTIAL       |

Select OFFERS on the left navigation. There is one credential offer from Registrar.

![image](tutorials/transfer/pictures/token-credentials-9.png)

Click ACCEPT.

Now a credential is created (Check CREDENTIALS on the left navigation).

![image](tutorials/transfer/pictures/token-credentials-10.png)

Congratulations! All credentials are ready. It is time for the token issuance and transfer.
