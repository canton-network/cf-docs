---
title: "ExternalPartyOnboardingDetails"
description: "Data class containing onboarding signed topology transactions for an external party. The constructor of this class ensures that only transactions related to the creation of an external party can be submitted. It performs validations on the kind of transactions and their relationship with each other. It does NOT validate anything related to the authorization of those transactions. That logic is implemented in the topology manager."
---
