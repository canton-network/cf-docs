---
title: "AcsCommitmentsCatchUpParameters"
description: "The class specifies the catch-up parameters governing the catch-up mode of a participant lagging behind with its ACS commitments computation. ***** Parameter recommendations A high catchUpIntervalSkip outputs more commitments and is slower to catch-up. For equal catchUpIntervalSkip, a high nrIntervalsToTriggerCatchUp is less aggressive to trigger the catch-up mode."
