The complete OpenAPI specification for the Scan API is published as a generated reference under [Splice APIs Scan API](/reference/splice-scan-api/common/get-readyz). It is organized by tag and includes the request and response schemas for every endpoint.

For an overview of the individual Scan API surfaces (current state, bulk data, aggregates, connectivity, operations, and CC reference data), see [API Overview Scan APIs](/sdks-tools/api-reference/splice-scan-api).

The raw OpenAPI source is maintained in the [splice repository](https://github.com/canton-network/splice/blob/main/apps/scan/src/main/openapi/scan.yaml).
