# Utility.Registry.V0.Configuration.AppReward

## Templates

<a id="type-utility-registry-v0-configuration-appreward-apprewardconfiguration-53465"></a>
**template** [AppRewardConfiguration](/docs-main/utilities/daml-api-reference/registry-model/Utility-Registry-V0-Configuration-AppReward#type-utility-registry-v0-configuration-appreward-apprewardconfiguration-53465)

> App reward configuration for distributing the app reward.
> Defines how the app reward is split between the operator and the provider. The operator’s weight
> `w` determines the fraction of the reward they receive; the remainder (1.0 - w) goes to the
> provider party by default. Note that the provider can optionally set explicit beneficiaries for
> their part of the share for certain instruments in the `InstrumentConfiguration`. If the
> configuration field is None, no app rewards will be distributed.
>
