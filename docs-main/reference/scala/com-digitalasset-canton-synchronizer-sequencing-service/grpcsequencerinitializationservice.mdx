---
title: "GrpcSequencerInitializationService"
description: "Initialize the Sequencer If the sequencer has not been previously initialized the request must either fail or start the sequencer and return successfully. The synchronizer will expect the SequencerService to be fully available when it receives a successful response and will immediately attempt to use it. If the request is received after the sequencer has been successfully initialized it should return successfully if the synchronizer_id matches the synchronizer that the sequencer has been initialized for, otherwise it should fail."
