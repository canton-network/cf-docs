```text
object Logging
```

**Summary**

The InstanceUUID is just a tag for com.daml.logging.LoggingContextOf to signal, that we request to include an UUID/ID so we can correlate related logs within a specific application run. A preferred instance of a com.daml.logging.LoggingContextOf containing a correlation id can be created via instanceUUIDLogCtx. TL:DR, Tag to request an Unique ID for an application run within logs.
