| `--target-postgres-tls-key file`            | Client’s private key (optional)<br/><br/>+ Environment variable: `SCRIBE_TARGET_POSTGRES_TLS_KEY`<br/>+ System property:      `target.postgres.tls.key`                                                                                                               |
