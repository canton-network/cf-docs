# Utility.Collateral.App.Model.State

## Templates

<a id="type-utility-collateral-app-model-state-collateralstate-5952"></a>

**template** [CollateralState]()

<a id="type-utility-collateral-app-model-state-collateralstate-5952"></a>

> Tracks the state of pledged collateral within a Collateral Agreement.
> This information is kept separate from the `CollateralAgreement` template in order
> to facilitate submission of transactions from external parties.

> Signatory: partyA, partyB, operator

> | Field               | Type                                                                                                                | Description                        |
> |---------------------|---------------------------------------------------------------------------------------------------------------------|------------------------------------|
> | partyA              | [Party](https://docs.digitalasset.com/build/3.3/reference/daml/stdlib/Prelude.html#type-da-internal-lf-party-57932) | The party for Party A in the CSA.  |
> | partyB              | [Party](https://docs.digitalasset.com/build/3.3/reference/daml/stdlib/Prelude.html#type-da-internal-lf-party-57932) | The party for Party B in the CSA.  |
> | operator            | [Party](https://docs.digitalasset.com/build/3.3/reference/daml/stdlib/Prelude.html#type-da-internal-lf-party-57932) | The Collateral App operator party. |
> | id                  | [Text](https://docs.digitalasset.com/build/3.3/reference/daml/stdlib/Prelude.html#type-ghc-types-text-51952)        | Collateral Agreement id.           |
> | collateralPositions | [[CollateralPosition](Utility-Collateral-App-Types.mdx#type-utility-collateral-app-types-collateralposition-89524)] | Collateral positions.              |
> + **Choice** Archive

>   Controller: partyA, partyB, operator

>   Returns: ()

>   (no fields)
