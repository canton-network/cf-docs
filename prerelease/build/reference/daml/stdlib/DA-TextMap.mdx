# DA.TextMap

TextMap - A map is an associative array data type composed of a
collection of key/value pairs such that, each possible key appears
at most once in the collection.

## Functions

<a id="function-da-textmap-fromlist-19033"></a>

[fromList]()
: : [([Text](Prelude.md#type-ghc-types-text-51952), a)] -> [TextMap](Prelude.md#type-da-internal-lf-textmap-11691) a
  <br/>
  Create a map from a list of key/value pairs.

<a id="function-da-textmap-fromlistwithl-22912"></a>

[fromListWithL]()
: : (a -> a -> a) -> [([Text](Prelude.md#type-ghc-types-text-51952), a)] -> [TextMap](Prelude.md#type-da-internal-lf-textmap-11691) a
  <br/>
  Create a map from a list of key/value pairs with a combining
  function. The combining function is only used when a key appears multiple
  times in the list and it takes two arguments: the first one is the new value
  being inserted at that key and the second one is the value accumulated so
  far at that key.
  Examples:
  <br/>
  ```daml
  >>> fromListWithL (++) [("A", [1]), ("A", [2]), ("B", [2]), ("B", [1]), ("A", [3])]
  fromList [("A", [3, 2, 1]), ("B", [1, 2])]
  >>> fromListWithL (++) [] == (empty : TextMap [Int])
  True
  ```

<a id="function-da-textmap-fromlistwithr-69626"></a>

[fromListWithR]()
: : (a -> a -> a) -> [([Text](Prelude.md#type-ghc-types-text-51952), a)] -> [TextMap](Prelude.md#type-da-internal-lf-textmap-11691) a
  <br/>
  Create a map from a list of key/value pairs like `fromListWithL`
  with the combining function flipped. Examples:
  <br/>
  ```daml
  >>> fromListWithR (++) [("A", [1]), ("A", [2]), ("B", [2]), ("B", [1]), ("A", [3])]
  fromList [("A", [1, 2, 3]), ("B", [2, 1])]
  >>> fromListWithR (++) [] == (empty : TextMap [Int])
  True
  ```

<a id="function-da-textmap-merge-26784"></a>

[merge]()
: : ([Text](Prelude.md#type-ghc-types-text-51952) -> a -> [Optional](Prelude.md#type-da-internal-prelude-optional-37153) c) -> ([Text](Prelude.md#type-ghc-types-text-51952) -> b -> [Optional](Prelude.md#type-da-internal-prelude-optional-37153) c) -> ([Text](Prelude.md#type-ghc-types-text-51952) -> a -> b -> [Optional](Prelude.md#type-da-internal-prelude-optional-37153) c) -> [TextMap](Prelude.md#type-da-internal-lf-textmap-11691) a -> [TextMap](Prelude.md#type-da-internal-lf-textmap-11691) b -> [TextMap](Prelude.md#type-da-internal-lf-textmap-11691) c
  <br/>
  Merge two maps. `merge f g h x y` applies `f` to all key/value pairs
  whose key only appears in `x`, `g` to all pairs whose key only appears
  in `y` and `h` to all pairs whose key appears in both `x` and `y`.
  In the end, all pairs yielding `Some` are collected as the result.
