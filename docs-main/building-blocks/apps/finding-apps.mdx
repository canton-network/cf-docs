The Canton Network ecosystem includes a growing number of applications across various categories. This page helps you discover and explore them.

## Where to Find Apps

### Canton Network Website

The primary directory of Canton Network applications.

**URL:** [canton.network](https://canton.network)

Browse featured applications and ecosystem partners.

### Global Synchronizer Foundation

Information about network participants and their applications.

**URL:** [sync.global](https://sync.global)

### Partner Directories

Ecosystem partners may maintain their own application directories.

## Evaluating Applications

When considering a Canton application:

### Technical Considerations

| Question | Why It Matters |
|----------|----------------|
| **What validator does it use?** | Your data will be hosted there |
| **Is it open source?** | Can you inspect the contracts? |
| **What APIs does it expose?** | How will you integrate? |
| **What standards does it follow?** | CIP compliance for interoperability |

### Trust Considerations

| Question | Why It Matters |
|----------|----------------|
| **Who operates it?** | Track record and reputation |
| **How long has it been running?** | Maturity and stability |
| **What audits has it passed?** | Security verification |
| **What support is available?** | Help when you need it |

### Privacy Considerations

| Question | Why It Matters |
|----------|----------------|
| **What data do you share?** | Understand visibility |
| **Who are the observers?** | Who else can see your data |
| **What's the data retention?** | How long is data kept |

## Using Canton Apps

### Typical Flow

1. **Connect** to the application (often via your validator)
2. **Authenticate** with your party/credentials
3. **Interact** with the application's features
4. **Manage** Canton Coin for transaction fees

### What to Expect

| Aspect | Expectation |
|--------|-------------|
| **Privacy** | Your data visible only to entitled parties |
| **Performance** | Transactions typically confirm in seconds |
| **Fees** | Traffic costs in Canton Coin |
| **Support** | Varies by application |
