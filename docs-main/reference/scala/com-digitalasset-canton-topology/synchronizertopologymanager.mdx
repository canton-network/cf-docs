---
title: "SynchronizerTopologyManager"
description: "Authorizes an existing topology transaction by signing it and adding it to the topology state. If expectFullAuthorization is true and the topology transaction cannot be fully authorized with keys from this node, returns with an error and the existing topology transaction remains unchanged."
