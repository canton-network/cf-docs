# DA.Functor

The `Functor` class is used for types that can be mapped over.

## Functions

<a id="function-da-functor-dollargt-48161"></a>

[($>)]()
: : [Functor](Prelude.md#class-ghc-base-functor-31205) f => f a -> b -> f b
  <br/>
  Replace all locations in the input (on the left) with the given
  value (on the right).

<a id="function-da-functor-ltampgt-91298"></a>

[(<&>)]()
: : [Functor](Prelude.md#class-ghc-base-functor-31205) f => f a -> (a -> b) -> f b
  <br/>
  Map a function over a functor. Given a value `as` and a function
  `f`, `as <&> f` is `f <$> as`. That is, `<&>` is like `<$>` but the
  arguments are in reverse order.

<a id="function-da-functor-ltdollardollargt-89503"></a>

[(<$$>)]()
: : ([Functor](Prelude.md#class-ghc-base-functor-31205) f, [Functor](Prelude.md#class-ghc-base-functor-31205) g) => (a -> b) -> g (f a) -> g (f b)
  <br/>
  Nested `<$>`.

<a id="function-da-functor-void-91123"></a>

[void]()
: : [Functor](Prelude.md#class-ghc-base-functor-31205) f => f a -> f ()
  <br/>
  Replace all the locations in the input with `()`.
