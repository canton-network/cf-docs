- Introduced: `3.4.10`
- Deprecated: -

```text
final case class RateLimitingConfig(maxApiServicesQueueSize: Int = 10000, maxApiServicesIndexDbQueueSize: Int = 1000, maxUsedHeapSpacePercentage: Int = 100, minFreeHeapSpaceBytes: Long = 0) extends Product with Serializable
```
