## `DynamicGrpcServer`

- Introduced: `3.4.10`
- Deprecated: -
- Removed: -

Upstream docs: [Open](https://docs.digitalasset.com/operate/3.4/scaladoc/com/digitalasset/canton/synchronizer/server/DynamicGrpcServer.html)

**Signature**

```text
class DynamicGrpcServer extends NamedLogging
```

**Summary**

Creates a dynamic public server to which services can be added at a later time, while providing a gRPC health service from the start. This is useful to bring up the sequencer node endpoint with health service, prior to being initialized.
