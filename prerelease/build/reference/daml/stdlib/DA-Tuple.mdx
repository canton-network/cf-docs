# DA.Tuple

Tuple - Ubiquitous functions of tuples.

## Functions

<a id="function-da-tuple-first-48871"></a>

[first]()
: : (a -> a’) -> (a, b) -> (a’, b)
  <br/>
  The pair obtained from a pair by application of a programmer
  supplied function to the argument pair’s first field.

<a id="function-da-tuple-second-48360"></a>

[second]()
: : (b -> b’) -> (a, b) -> (a, b’)
  <br/>
  The pair obtained from a pair by application of a programmer
  supplied function to the argument pair’s second field.

<a id="function-da-tuple-both-63511"></a>

[both]()
: : (a -> b) -> (a, a) -> (b, b)
  <br/>
  The pair obtained from a pair by application of a programmer
  supplied function to both the argument pair’s first and second
  fields.

<a id="function-da-tuple-swap-76115"></a>

[swap]()
: : (a, b) -> (b, a)
  <br/>
  The pair obtained from a pair by permuting the order of the
  argument pair’s first and second fields.

<a id="function-da-tuple-dupe-14430"></a>

[dupe]()
: : a -> (a, a)
  <br/>
  Duplicate a single value into a pair.
  <br/>
  > dupe 12 == (12, 12)

<a id="function-da-tuple-fst3-84676"></a>

[fst3]()
: : (a, b, c) -> a
  <br/>
  Extract the ‘fst’ of a triple.

<a id="function-da-tuple-snd3-63950"></a>

[snd3]()
: : (a, b, c) -> b
  <br/>
  Extract the ‘snd’ of a triple.

<a id="function-da-tuple-thd3-58697"></a>

[thd3]()
: : (a, b, c) -> c
  <br/>
  Extract the final element of a triple.

<a id="function-da-tuple-curry3-2900"></a>

[curry3]()
: : ((a, b, c) -> d) -> a -> b -> c -> d
  <br/>
  Converts an uncurried function to a curried function.

<a id="function-da-tuple-uncurry3-51859"></a>

[uncurry3]()
: : (a -> b -> c -> d) -> (a, b, c) -> d
  <br/>
  Converts a curried function to a function on a triple.
