---
title: "CantonConfigValidator"
description: "Type class for validations of Canton config classes depending on CantonEditions. Instances are typically derived using manual.CantonConfigValidatorDerivation via Magnolia so that validation automatically calls validate on the subconfigurations, where the subconfigurations of A are all the types of fields of a case class that implements A. The derivation must find a corresponding CantonConfigPrevalidator instance via implicit resolution."
---
