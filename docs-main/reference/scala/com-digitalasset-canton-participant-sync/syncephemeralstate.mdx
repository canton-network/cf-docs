title: "SyncEphemeralState"
description: "The participant-relevant state and components for a synchronizer that is kept only in memory and must be reconstructed after crashes and fatal errors from the com.digitalasset.canton.participant.store.SyncPersistentState. The ephemeral state can be kept across network disconnects provided that the local processing continues as far as possible."
