# Rotate keys with KMS

Canton supports both [rotating the wrapper key](rotate_wrapper_key.md#rotate-wrapper-key) when using envelope encryption,
and [rotating Canton keys stored externally in a KMS](rotate_external_keys.md#rotate-external-keys).

For smooth disaster recovery, key rotation must be interleaved with backups as described under [Backup and Restore](../../../recover/backup-restore.md#backup-restore-private-key-rotation).

* [Rotate the envelope wrapper key](rotate_wrapper_key.md)
* [Rotate external KMS keys](rotate_external_keys.md)
