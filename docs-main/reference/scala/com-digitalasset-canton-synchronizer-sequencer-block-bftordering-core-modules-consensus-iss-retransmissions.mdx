### `EpochStatusBuilder`

- Introduced: `3.4.10`
- Deprecated: -
- Removed: -

Upstream docs: [Open](https://docs.digitalasset.com/operate/3.4/scaladoc/com/digitalasset/canton/synchronizer/sequencer/block/bftordering/core/modules/consensus/iss/retransmissions/EpochStatusBuilder.html)

**Signature**

```text
class EpochStatusBuilder extends AnyRef
```

**Summary**

As part of retransmissions, we broadcast an epoch status to other nodes. However, because we have separate modules processing each segment, we need to request and get segment statuses from each segment first in order to build the epoch status. This helper class is used to process (potentially out-of-order) segment status messages and then build the epoch status once all segment status messages have arrived.
