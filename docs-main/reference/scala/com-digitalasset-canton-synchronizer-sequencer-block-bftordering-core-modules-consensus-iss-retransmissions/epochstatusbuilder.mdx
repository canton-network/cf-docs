---
title: "EpochStatusBuilder"
description: "As part of retransmissions, we broadcast an epoch status to other nodes. However, because we have separate modules processing each segment, we need to request and get segment statuses from each segment first in order to build the epoch status. This helper class is used to process (potentially out-of-order) segment status messages and then build the epoch status once all segment status messages have arrived."
---
