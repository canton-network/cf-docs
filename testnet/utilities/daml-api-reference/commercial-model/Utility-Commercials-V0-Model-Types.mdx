> > | lastBilledAt | [Time](https://docs.digitalasset.com/build/3.3/reference/daml/stdlib/Prelude.html#type-da-internal-lf-time-63886) | Ledger time when this fee was last billed.                                                                                                                      |
> > | billedUntil  | [Time](https://docs.digitalasset.com/build/3.3/reference/daml/stdlib/Prelude.html#type-da-internal-lf-time-63886) | The time until which the fee is paid for. The fee is billed in integer multiples of the billing period, i.e., `billedUntil = lastBilledAt + N x billingPeriod`. |
