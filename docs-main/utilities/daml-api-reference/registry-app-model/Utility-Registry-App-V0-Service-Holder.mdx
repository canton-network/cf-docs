# Utility.Registry.App.V0.Service.Holder

Deprecated: This module will be deactivated in an upcoming release of this Daml package.
For mint, burn, and transfer operations, use the recommended two-step flows
initiated on the `Utility.Registry.App.V0.Service.AllocationFactory.AllocationFactory` contract.
The lock and unlock have been disabled as of version `0.6.0` of this package.
