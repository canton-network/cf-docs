---
title: "EngineController"
description: "Simple class to embody the connection between a component that wants to abort an engine computation and the engine itself. An instance of this class is created by the protocol processor for each request and passed along during processing. When a condition determines that the computation should be aborted, the component calls abort. The instance is also passed to the engine when performing a computation. During the execution of a ResultInterruption, the engine will call abortStatus to determine whether it should abort or continue."
