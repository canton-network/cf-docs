# The Standard Library

The Daml standard library is a collection of Daml modules that are bundled with the SDK, and can be used to implement Daml applications.

The [Prelude](Prelude.md#module-prelude-72703) module is imported automatically in every Daml module. Other modules must be imported manually, just like your own project’s modules. For example:

```daml
import DA.Optional
import DA.Time
```
