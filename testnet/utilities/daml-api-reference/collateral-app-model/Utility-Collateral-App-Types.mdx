**data** [EligibleInstrument]()

<a id="type-utility-collateral-app-types-eligibleinstrument-214"></a>

> We do not use `InstrumentId` directly for extensibility. We might add e.g. haircuts in
> a future iteration of the app. `InstrumentId` is fixed by the standard.
