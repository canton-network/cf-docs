---
title: "EnterpriseMediatorReplicaManager"
description: "Manages replicas of a single Mediator instance. Passive instances are currently entirely passive and have no components running at runtime. When becoming active a MediatorRuntime is started and connected to the admin services. When becoming passive the running MediatorRuntime is shutdown and the admin services are disconnected."
