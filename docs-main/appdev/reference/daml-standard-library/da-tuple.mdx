## Functions

<span id="function-da-tuple-first-48871"></span>

### `first`

`first` : (`a` -> `a'`) -> (`a`, `b`) -> (`a'`, `b`)

The pair obtained from a pair by application of a programmer
supplied function to the argument pair's first field.

<span id="function-da-tuple-second-48360"></span>

### `second`

`second` : (`b` -> `b'`) -> (`a`, `b`) -> (`a`, `b'`)

The pair obtained from a pair by application of a programmer
supplied function to the argument pair's second field.

<span id="function-da-tuple-both-63511"></span>

### `both`

`both` : (`a` -> `b`) -> (`a`, `a`) -> (`b`, `b`)

The pair obtained from a pair by application of a programmer
supplied function to both the argument pair's first and second
fields.

<span id="function-da-tuple-swap-76115"></span>

### `swap`

`swap` : (`a`, `b`) -> (`b`, `a`)

The pair obtained from a pair by permuting the order of the
argument pair's first and second fields.

<span id="function-da-tuple-dupe-14430"></span>

### `dupe`

`dupe` : `a` -> (`a`, `a`)

Duplicate a single value into a pair.

> dupe 12 == (12, 12)

<span id="function-da-tuple-fst3-84676"></span>

### `fst3`

`fst3` : (`a`, `b`, `c`) -> `a`

Extract the 'fst' of a triple.

<span id="function-da-tuple-snd3-63950"></span>

### `snd3`

`snd3` : (`a`, `b`, `c`) -> `b`

Extract the 'snd' of a triple.

<span id="function-da-tuple-thd3-58697"></span>

### `thd3`

`thd3` : (`a`, `b`, `c`) -> `c`

Extract the final element of a triple.

<span id="function-da-tuple-curry3-2900"></span>

### `curry3`

`curry3` : ((`a`, `b`, `c`) -> `d`) -> `a` -> `b` -> `c` -> `d`

Converts an uncurried function to a curried function.

<span id="function-da-tuple-uncurry3-51859"></span>

### `uncurry3`

`uncurry3` : (`a` -> `b` -> `c` -> `d`) -> (`a`, `b`, `c`) -> `d`

Converts a curried function to a function on a triple.
