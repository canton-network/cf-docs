<a id="function-da-text-dropsuffix-37682"></a>

[dropSuffix]()
: : [Text](Prelude.mdx#type-ghc-types-text-51952) -> [Text](Prelude.mdx#type-ghc-types-text-51952) -> [Text](Prelude.mdx#type-ghc-types-text-51952)
  <br/>
  Drops the given suffix from the argument. It returns the original
  text if the text doesn’t end with the given suffix. Examples:
  <br/>
  ```daml
  dropSuffix "!" "Hello World!"  == "Hello World"
  dropSuffix "!" "Hello World!!" == "Hello World!"
  dropSuffix "!" "Hello World."  == "Hello World."
  ```

<a id="function-da-text-stripsuffix-58624"></a>

[stripSuffix]()
: : [Text](Prelude.mdx#type-ghc-types-text-51952) -> [Text](Prelude.mdx#type-ghc-types-text-51952) -> [Optional](Prelude.mdx#type-da-internal-prelude-optional-37153) [Text](Prelude.mdx#type-ghc-types-text-51952)
  <br/>
  Return the prefix of the second text if its suffix matches the
  entire first text. Examples:
  <br/>
  ```daml
  stripSuffix "bar" "foobar" == Some "foo"
  stripSuffix ""    "baz"    == Some "baz"
  stripSuffix "foo" "quux"   == None
  ```

<a id="function-da-text-parsenumeric-9858"></a>

[parseNumeric]()
: : [NumericScale](Prelude.mdx#class-ghc-classes-numericscale-83720) n => [Text](Prelude.mdx#type-ghc-types-text-51952) -> [Optional](Prelude.mdx#type-da-internal-prelude-optional-37153) ([Numeric](Prelude.mdx#type-ghc-types-numeric-891) n)
  <br/>
  Attempt to parse a `Numeric` value from a given `Text`.
  To get `Some` value, the text must follow the regex
  `(-|\+)?[0-9]+(\.[0-9]+)?`
  In particular, the shorthands `".12"` and `"12."` do not work,
  but the value can be prefixed with `+`.
  Leading and trailing zeros are fine, however spaces are not.
  Examples:
  <br/>
  ```daml
  parseNumeric "3.14" == Some 3.14
  parseNumeric "+12.0" == Some 12
  ```
