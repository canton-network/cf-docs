---
title: "AuthenticationTokenManager"
description: "Attempts to hold a valid authentication token. The first token will not be fetched until getToken is called for the first time. Subsequent calls to getToken before the token is obtained will be resolved for the first token. getToken always returns a EitherT[Future, ...] but if a token is already available will be completed immediately with that token."
